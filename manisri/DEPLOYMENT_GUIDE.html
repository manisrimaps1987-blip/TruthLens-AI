GitHub Deployment - Interactive Guide

✅ Step completed! Click next to continue.
1

Install Git (If Not Installed)

Git is required for version control and GitHub deployment.

  • Download Git from git-scm.com
  • Run the installer with default settings
  • Restart PowerShell/Terminal
  • Verify: git --version
📥 Download Git
2

Create GitHub Account & Repository

Go to GitHub and create a new repository for TruthLens AI.

⚠️ Important: When creating the repository, select "Public" and choose "MIT License"
Repository Name: TruthLensAI Description: Intelligent Misinformation Detection System Public: ✓ YES License: MIT License Add README: NO (we have one)
🔗 Create Repository
3

Create Personal Access Token

Create a token for authentication instead of using your password.

  • Go to GitHub Settings → Developer settings → Personal access tokens
  • Click "Generate new token (classic)"
  • Name: "TruthLens AI Deployment"
  • Select "repo" scope (full control of repositories)
  • Copy the token and save it securely
⚠️ Important: You can only see the token once! Copy it immediately after generation.
🔐 Generate Token
4

Configure Git Locally

Set up your Git configuration with your GitHub information.

git config --global user.name "Your Full Name" git config --global user.email "your.email@github.com"

Replace with your actual name and GitHub email address

5

Initialize & Commit

Initialize Git in your project folder and create the first commit.

cd "c:\Users\manis\OneDrive\Desktop\manisri\TruthLensAI" git init git add . git commit -m "Initial commit: TruthLens AI v1.0.0"
6

Connect to GitHub

Connect your local repository to GitHub. Replace YOUR_USERNAME with your GitHub username.

git branch -M main git remote add origin https://github.com/YOUR_USERNAME/TruthLensAI.git git push -u origin main
⚠️ When prompted for password, use your Personal Access Token (not your GitHub password)
7

Deploy to Streamlit Cloud (Optional)

Deploy your app for free with Streamlit Cloud.

  • Go to Streamlit Cloud website
  • Click "New app"
  • Connect your GitHub account
  • Select: YOUR_USERNAME / TruthLensAI / demo_app.py
  • Click "Deploy"
☁️ Streamlit Cloud
